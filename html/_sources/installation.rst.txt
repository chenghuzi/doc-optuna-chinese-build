安装
============

Optuna 支持 3.5 或者更新版本的 Python。

我们推荐使用 pip 来安装 Optuna：

.. code-block:: bash

    $ pip install optuna

你也可以在 Git 仓库的 master 分支下用开发模式安装Optuna：

.. code-block:: bash

    $ pip install git+https://github.com/optuna/optuna.git

你也可以通过 conda 来安装 Optuna：

.. code-block:: bash

    $ conda install -c conda-forge optuna
